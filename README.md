Laravel Nostr
====

**Work in progress**

Depends on this WebAPI.  
https://github.com/kawax/nostr-vercel-api

## Requirements
- PHP >= 8.1
- Laravel >= 9.0

## Installation

```shell
composer require revolution/laravel-nostr

# composer require revolution/laravel-nostr:dev-main

php artisan vendor:publish --tag=nostr-config
```
